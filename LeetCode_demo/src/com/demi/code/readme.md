### 回溯
 可以用到回溯的题型：
    排列、组合、切割、子集、棋盘等等。能够形成树结构的形式。
    --->对于树的代码处理：横向对应for循环；纵向对应递归
    
    void backtracking(参数) {
        if (终止条件) {
            收集结果；
            return;
        }
        for (集合元素) {
            处理节点;
            递归函数;
            回溯操作...
        }
        return;
    }
